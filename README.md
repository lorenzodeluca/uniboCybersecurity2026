# uniboCybersecurity2026

In this repository, I will test some Java classes for cybersecurity, such as SecureRandom.

## Overview

This project focuses on testing and demonstrating the usage of various Java security classes to ensure they are implemented correctly for cybersecurity purposes.

## Goals

- Validate the functionality of cryptographic classes
- Test randomness and entropy in SecureRandom
- Ensure proper encryption and decryption with Cipher
- Verify hashing algorithms in MessageDigest
- Demonstrate secure coding practices

## Classes to Test

- SecureRandom: For generating cryptographically strong random numbers
- MessageDigest: For creating hash values (SHA-1, SHA-256)
- Cipher: For encryption and decryption operations
- KeyPairGenerator: For generating public/private key pairs
- Signature: For digital signatures

## Setup

1. Ensure Java 8 or higher is installed
2. Clone the repository
3. Compile the Java files using `javac`
4. Run the test classes

## Usage

Run individual test classes to see examples of each security class in action.
